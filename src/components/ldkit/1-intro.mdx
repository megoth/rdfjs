<Content>
    LDkit is Linked Data query toolkit for TypeScript developers.

    LDkit lets you access, display and modify any RDF data, and it works in browser, [Deno](https://deno.land/) and
    [Node](https://nodejs.org/). It is sort of an ORM for graph databases.

    <h2 className="subtitle">Typical use</h2>

    The typical LDkit workflow to access Linked Data comprises of these protocols:

    - You define a data source and other settings through [Options](https://ldkit.io/docs/components/options).
    - You define a [Namespace](https://ldkit.io/docs/components/namespaces) for the data to be retrieved, or use an
    existing one.
    - You define a data [Schema](https://ldkit.io/docs/components/schema) with the help of namespaces.
    - You create a data [Lens](https://ldkit.io/docs/components/lens) and pass it the schema and options to query data.
    - Optionally, you can tweak the [Query Engine](https://ldkit.io/docs/components/query-engine).
</Content>
